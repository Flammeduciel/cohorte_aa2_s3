Bienvenue dans le hub

La Cohorte 2 réunit environ 200 académiciens aux profils variés : développeurs, designers, analystes et entrepreneurs. Ce hub est notre vitrine collective. On n'attend pas de validation : chacun vient indexer son projet ici pour le partager avec la promo.

Chaque semaine, ce site évolue avec nous. Les projets s'accumulent, les liens se mettent à jour, les membres progressent. Tout ce qui compte pour la cohorte se retrouve ici.
